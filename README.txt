# Hướng dẫn chạy dự án nhanh

1. Mở PowerShell ở thư mục gốc dự án
2. Gõ: .\run.ps1
   (script sẽ bypass ExecutionPolicy, chạy backend + frontend, build production)
3. Nếu muốn tạo file ZIP final thủ công, gõ: .\build_zip.bat

Backend chạy tại: http://localhost:4000
Frontend chạy tại: http://localhost:1573
